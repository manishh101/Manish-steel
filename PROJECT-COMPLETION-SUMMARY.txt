🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────

ISSUE: "vercel command not found"
  SOLUTION: npm install -g vercel

ISSUE: "Database connection failed"
  SOLUTION: Check MongoDB URI, whitelist IPs in Atlas

ISSUE: "CORS errors"
  SOLUTION: Add frontend URL to ALLOWED_ORIGINS

ISSUE: "Environment variables not set"
  SOLUTION: Set in Vercel Dashboard, then redeploy
